================================================================================
故障排查
================================================================================

问题: 训练时报错 KeyError: 'auxiliary_targets'
解决: 检查配置 enable_auxiliary_tasks: true

问题: 某个辅助任务损失为NaN
解决: 降低学习率或增加warmup_epochs

问题: 不确定性权重震荡
解决: 使用固定权重 use_uncertainty_weighting: false

================================================================================
技术原理
================================================================================

辅助任务标签推导:
  valence = 1.0 - (depression / 3.0)  # 愉悦度
  arousal = anxiety / 3.0              # 激活度

  emotion_cls:
    depression > 1.5 → 悲伤
    stress > 1.5     → 愤怒
    all_normal       → 快乐
    else             → 中性

不确定性加权:
  总损失 = Σ [ (1/2σ²) × L_i + log(σ) ]

  σ²大 → 不确定性高 → 权重小
  σ²小 → 确定性高   → 权重大
